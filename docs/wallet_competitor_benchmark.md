# Wallet Competitor Benchmark

## Goal
Analyze leading wallet and DEX products and translate their strongest UX and
product patterns into Acorus Wallet without copying branding or code.

## Products
- MetaMask
- Trust Wallet
- Uniswap Wallet
- PancakeSwap

## Feature matrix

| Product | Primary strength | Multichain posture | dApp / extension posture | UX patterns to learn |
| --- | --- | --- | --- | --- |
| MetaMask | Web3 identity + transaction control | Strong EVM-first network switching | Browser extension, connected sites, permissions, confirmations | Review screens, permission model, advanced controls, activity |
| Trust Wallet | Mass-market multichain wallet | 100+ chains / millions of assets positioning | Mobile + extension + Web3 access | Beginner language, action cards, multichain hub, security education |
| Uniswap Wallet | Swap-first self-custody wallet | 19+ networks with routing focus | Explore apps later, but wallet UX centers on swap/send/bridge | Premium minimal layout, big numbers, token search, clear route UX |
| PancakeSwap | All-in-one DEX ecosystem | Multichain trade and earn framing | Ecosystem entrypoint more than wallet identity | Explore / Earn / Quests / Launch / Trending sections, gamified discovery |

## MetaMask patterns

### Strengths
- Browser extension as primary Web3 identity
- dApp connect
- Permission review
- Transaction confirmation
- Network switching
- Token import
- Swap
- Portfolio
- Activity history
- Advanced settings

### What Acorus should adopt
- Extension-ready architecture
- Connected sites and permissions shell
- Transaction review model
- Network and account selector
- Advanced transaction controls later
- Risk warnings

## Trust Wallet patterns

### Strengths
- Beginner-friendly multichain wallet
- Mobile + extension
- dApp browser
- Swaps
- Staking
- NFTs
- Buy crypto
- Security education

### What Acorus should adopt
- Simple action cards
- Multichain asset hub
- Security Center
- Backup reminders
- Explore Web3 hub
- Token discovery

## Uniswap Wallet patterns

### Strengths
- Swap-first design
- Minimal premium UI
- Large balance presentation
- Clear action buttons
- Token search
- Favorite tokens and wallets
- Bridge / send / swap / explore

### What Acorus should adopt
- Premium dashboard
- Swap-first UX language where useful
- Token, favorite, and watch-wallet concepts
- Clean route preview
- Search-first token UX

## PancakeSwap patterns

### Strengths
- DEX ecosystem
- Swap
- Liquidity
- Farms
- Pools
- Prediction
- Lottery
- Launchpad / springboard
- Gamified DeFi

### What Acorus should adopt
- Explore sections:
  - Trade
  - Earn
  - Quests
  - Launch
  - Trending
  - Meme radar
- Gamified onboarding
- DeFi cards
