# Product UX Upgrade Plan

## Goal
Upgrade Acorus Wallet UI using the best patterns from leading wallets and DEXes
while keeping its own universal multichain identity.

## UX principles
- One wallet, many networks
- Big balance first
- Action cards second
- Risk status always visible
- Beginner-friendly default
- Advanced controls hidden but available
- Every disabled feature explains why
- No fake live DeFi

## Immediate UI upgrades
- Dashboard action grid
- Explore shell
- Security Center shell
- Extension readiness card
- Product feature cards
