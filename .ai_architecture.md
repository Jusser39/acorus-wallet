# AI Architecture Notes: Acorus Wallet

**Overview:**
Acorus Wallet is a multichain Web3 wallet supporting EVM, Solana, Bitcoin, and TON. It consists of a Next.js web application (`apps/web`), a browser extension (`apps/extension`), and a backend API proxy (`apps/api`).

**Key Technologies:**
- **Frontend**: Next.js (App Router), React, Tailwind CSS, Zustand (state management).
- **Core Library**: `@acorus/wallet-core` handles encryption, vaults, and transaction building.
- **Shared Utilities**: `@acorus/shared` contains cross-chain types and formatting functions.
- **APIs**: Jupiter (Solana swaps), Rango (Cross-chain), 0x (EVM swaps).

**Important Web Components:**
- `SwapComposer` (`apps/web/components/swap-composer.tsx`): The main UI for routing and fetching quotes. Relies on dummy zero-addresses for unauthorized indicative quotes.
- `AppBootstrap` (`apps/web/components/app-bootstrap.tsx`): Handles initial local storage decryption and wallet profile loading.
- `WalletStore` (`apps/web/store/wallet-store.ts`): Zustand store managing the active profile, theme, and encrypted vault state.
