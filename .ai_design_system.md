# AI Design & UI Memory: Acorus Wallet

**CRITICAL UI RULES:**
1. **Localization**: Unless otherwise specified, dynamically added UI strings (especially for new features or user-facing alerts) should prioritize Russian translations, as requested by the user (e.g., "Из (From)", "В (To)", "Обменный курс").
2. **Aesthetics**: The app uses a highly polished "Glassmorphism" aesthetic.
   - Use utility classes like `app-surface`, `premium-card`, and `subtle-grid`.
   - Primary buttons should use `button-primary` (usually rounded with gradients).
   - Secondary buttons should use `button-secondary`.
3. **Animations**: Interactions must be smooth. Buttons should have `transition hover:-translate-y-0.5` or similar micro-animations.
4. **Layout**: Ensure responsive design. Slide-out panels (like token pickers) should be absolutely positioned and pop out to the LEFT (`xl:right-full`) to avoid overlapping main content like the Portfolio.
